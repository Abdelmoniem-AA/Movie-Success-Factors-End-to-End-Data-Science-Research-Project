# Movie Success Factors

An independent data science research project investigating the impact of production budget, cast popularity, audience ratings, critics' ratings, and release season on movie box office revenue.

## Project Overview

This study analyzes a dataset of 5,400+ movies collected from IMDb, Box Office Mojo, TMDb, and Kaggle sources. The objective is to identify the strongest predictors of movie revenue and uncover patterns related to release timing and audience reception.

## Dataset

- 5,400+ movie records
- 13 features
- Sources:
  - IMDb
  - Box Office Mojo
  - TMDb
  - Kaggle

## Methods

- Data collection and integration
- Missing value imputation
- Feature engineering
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Random Forest Regression
- Revenue Classification Modeling
- K-Means Clustering

## Key Findings

- Production budget was the strongest predictor of revenue (r = 0.71).
- IMDb ratings showed only a weak relationship with revenue (r = 0.19).
- Summer releases achieved the highest average box-office revenue.
- Critics' and audience ratings demonstrated strong agreement (r = 0.64).
